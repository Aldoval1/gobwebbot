## 2024-05-18 - Semantic Map Zoom Controls\n**Learning:** Custom interactive UI elements like map zoom controls are sometimes implemented as `<div>` elements with `onclick` handlers, completely breaking keyboard accessibility and screen reader support.\n**Action:** When encountering custom controls, refactor them to use semantic `<button type='button'>` tags. Always add descriptive `aria-label` attributes for icon-only buttons, `aria-hidden='true'` to decorative inner icons, and ensure explicit `:focus-visible` CSS rules are in place for keyboard navigability.
